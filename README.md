#### html 压缩
- HTML 代码压缩就是压缩这些在文本文件中有意义，但是在 HTML 中 **不显示**的字符，包括**空格**，**制表符**，**换行符**等，还有一些其它意义的字符，如 **HTML 注释**也可以被压缩。
如何进行压缩：
1. 使用在线网站进行压缩
2. nodejs 提供了 html-minifier 工具
3. 后端模板引擎渲染压缩
